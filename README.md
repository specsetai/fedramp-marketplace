Public information and machine-readable metadata supporting Specset's **Initial Implementation** listing on the [FedRAMP Marketplace](https://www.fedramp.gov/20x/).

- **Provider:** Specset
- **Offering:** Specset Cloud Platform
- **Website:** https://specset.com
- **Trust Center:** https://trust.specbook.ai
- **Current stage:** Initial Implementation
- **Target framework:** FedRAMP 20x Class A
- **Federal use case:** Indirect Use — Specset is intended to be included as a third-party information resource within other cloud service offerings operated by FedRAMP-certified providers.

## About this repository

This repository hosts human-readable and machine-readable public information supporting Specset's marketplace listing request. The authoritative machine-readable standard for FedRAMP public offering information (`CDS-CSO-PUB`) is FedRAMP's own published JSON Schema / OSCAL — see [github.com/FedRAMP/rules](https://github.com/FedRAMP/rules). `fedramp.json` is an intent placeholder and will be replaced with a schema-conformant package ahead of Class A submission.
